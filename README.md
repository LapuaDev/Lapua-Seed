<h1>Інструкція Lapua Seed</h1>

Прикладіть керно у відповідне місце та вдарте по ньому молотком щоб отримати глибоку відмітку. Для зручності, перед нанесення заглиблень керном, зробіть позначки маркером і перевірте на помилки.

<h2>Як записати фразу</h2>
Кожен рядок на пластині призначений для одного слова seed фрази. Записується не саме слово в текстовому форматі, а його порядковий номер у словнику BIP39 розкладений на доданки послідовності степенів числа 2. Кодування слів можна знайти у каталозі [BIP39](bip39.md) або навчитись <a href="https://lapua.dev/calculate.html" target="_blank"> вираховувати самостійно</a>.
<br><br>
Приклад пластини для seed фрази з 12 слів у котрій перше слово <b>ghost</b>, п'яте <b>avocado</b>, а останнє <b>behind</b>.

<h2>Як прочитати Seed фразу</h2>
Потрібно додати цифри у рядку навпроти котрих стоять відмітки. Отримана сума - це порядковий номером слова у каталозі BIP39.
